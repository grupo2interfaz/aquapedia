# AquaPedia 🌊

Enciclopedia digital de la vida acuática.

## Descripción

AquaPedia es un sitio web estático desarrollado como Trabajo Práctico N°1 para la materia **Tecnologías para el Desarrollo de Interfaces Digitales** (Universidad de Belgrano, 1° cuatrimestre 2026).

## Páginas

| Archivo | Descripción |
|---|---|
| `index.html` | Página de inicio — Hero, estadísticas, grilla de categorías y CTA |
| `sobre-nosotros.html` | Amenazas oceánicas, datos, formulario de voluntariado |
| `mamiferos.html` | Página interna: Mamíferos Marinos |
| `explorar.html` | Grilla de todas las categorías |

## Tecnologías

- HTML5 semántico
- CSS3 (variables, grid, flexbox, responsive)
- JavaScript vanilla (menú mobile)
- Google Fonts: Playfair Display + DM Sans

## Cómo visualizar

Abrí `index.html` en tu navegador, o usá la extensión **Live Server** de VS Code para verlo con recarga automática.
